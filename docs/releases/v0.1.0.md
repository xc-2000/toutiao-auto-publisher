# Toutiao Auto Publisher v0.1.0

首个公开版本。Toutiao Auto Publisher 是面向头条创作者的本地 AI 内容工作台，可聚合实时热点，生成图文、封面和视频，管理多账号自动化，并关联创作奖励活动完成发布。

## 主要功能

- 聚合头条、百度、微博、知乎、抖音、B站等平台的实时热榜，支持去重、分类和查询。
- 使用 OpenAI-compatible 模型生成标题、正文、标签、封面和视频。
- 发现进行中的创作活动，解析任务规则、参与周期和奖励信息。
- 按热点相关度匹配活动，支持图文 HTTP 协议和视频 VOD/TOS 协议发布。
- 支持多用户数据隔离、多头条账号和独立自动化配置。
- 提供内容库、生成进度、失败恢复、编辑预览和分页管理。

## 快速开始

```powershell
git clone https://github.com/xc-2000/toutiao-auto-publisher.git
cd toutiao-auto-publisher
py -3.11 -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
npm ci
Copy-Item config.example.toml config.toml
$env:OPENAI_API_KEY = "YOUR_API_KEY"
python dashboard.py --config config.toml
```

完整配置、Linux/macOS 启动方法和使用说明请查看 [README](https://github.com/xc-2000/toutiao-auto-publisher#readme)。

## 数据安全

账号 Cookie、模型密钥、应用用户数据、日志和生成媒体默认保存在本机并已加入 `.gitignore`，请勿将本地状态目录提交到公开仓库。
